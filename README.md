# Weather
Open Weather (Node, Express, EJS, REST)
Simple weather app that currently has two features:
- Current temperature for a given city
- Forecast for five days

Tech:
- Nodejs
- Expressjs
- EJS
- Back end
  - REST API (OpenWeather API)
  - https://openweathermap.org/api
